# 🔍 Network Port Scanner (Python GUI)

<p align="center">
  <b>Fast • Simple • Beginner-Friendly Port Scanner</b>
</p>

---

## 🚀 Features

- 🖥️ GUI built with Tkinter  
- ⚡ Multi-threaded scanning (fast performance)  
- 🎯 Custom port range (1–65535)  
- 📡 Real-time scan results  
- 💾 Save results (TXT / CSV)  
- 🛑 Stop scan anytime  

---

## 🛠️ Requirements

- Python 3.7 or above  
- Tkinter (comes pre-installed)

---

## 📦 Installation

```bash
git clone https://github.com/Venkat-S777/Network_Port_Scanner_GUI.git
cd Network_Port_Scanner_GUI
```

---

▶️ Run the Application
```
python scanner.py
```

---

🧭 How to Use
---
- Enter target IP or hostname
  Example: 127.0.0.1 or google.com
- Enter port range
  Example: 1 to 1024
- Click Start Scan
- View results in real-time
- Save results if needed

---


🔎 Scan Output
---
The scanner shows:
- Port number
- Service name (if available)
- Port status (Open / Closed)
- Response time
